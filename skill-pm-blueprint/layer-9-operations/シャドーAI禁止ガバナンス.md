# シャドーAI禁止ガバナンス

## 品質ゲート対応
- **GOV-SHADOW-AI-001** (critical): 会社未承認AIツール使用禁止方針

## 概要

シャドー AI (Shadow AI) = **会社が承認していない AI ツールを社員が個人的に業務で利用すること**。
個人情報・機密情報の意図せぬ流出、ハルシネーション結果の業務利用、ライセンス違反などのリスクをもたらす。

## いつ使うか

- 矢羽① 業務要件定義 第1月で全社ポリシー策定
- 矢羽⑨ 要員育成実行 (4,500名 CS 要員教育) と連動
- 本番運用以降、定期的に違反検知・教育

## 主要リスク

| シナリオ | リスク内容 |
|---------|-----------|
| OPが応対中に ChatGPT に顧客情報を貼り付けて要約させる | PII 漏洩 (個情法第18条違反) |
| 営業支援が無料 Copilot で報告書を作成、不正確な内容をそのまま顧客提出 | ハルシネーション → 顧客信用失墜 |
| エンジニアが Claude にコードを貼り付けてレビュー依頼、自社知財流出 | 知財漏洩 |
| HR が Gemini で人事評価データを分析、学習データに使用される | 個人情報目的外利用 |
| 個人が DeepSeek 等の海外モデルで業務データ処理 | 越境データ移転 (個情法第28条) |

## 禁止/許可方針 (4区分)

### A. 完全禁止
- 顧客個人情報・機微情報を含む業務データの入力
- 海外リージョンに学習データとして送信される無料/個人版 AI ツール
- 経営機密 (経営計画/M&A/未公開財務情報) の入力

### B. 条件付き許可
- 一般情報の翻訳・要約・アイデア出し: 社内承認 AI (Azure OpenAI 等) のみ
- コード生成: GitHub Copilot Business (社内アカウント) のみ
- 顧客向け報告書ドラフト: UC-012 の社内システム + 人手レビュー必須

### C. 推奨
- 社内承認 AI ツール: Azure OpenAI / GitHub Copilot Business / Microsoft 365 Copilot
- これらは契約で「学習に使用しない」「データは国内リージョン保管」を明記

### D. 評価中
- 新しい AI ツールが出てきた場合、AI 倫理委員会 + セキュリティ + 法務の三者で評価後許可判断

## ポリシー文書テンプレ

```markdown
# ABC社 AI ツール利用ポリシー (v1.0)

## 1. 適用範囲
全社員 (4,500 名 CS 要員 + 営業支援要員 + 本社企画 + 開発委託先)

## 2. 承認 AI ツール一覧
- Azure OpenAI Service (社内アカウント経由)
- GitHub Copilot Business (社内ライセンス)
- Microsoft 365 Copilot
- (※随時更新)

## 3. 禁止事項
- 個人版 ChatGPT / Claude / Gemini / DeepSeek 等への業務データ入力
- 顧客個人情報・録音テキスト・配達データの外部 AI への入力
- 経営機密の AI への入力

## 4. 違反時
- 初回: 教育受講
- 再犯: 懲戒委員会
- 重大事案: 解雇 + 法的措置検討

## 5. 同意・誓約
本ポリシーに従うことに同意します:
社員番号: __________  氏名: __________  日付: __________
```

## 教育プログラム

### 全社員必須 (年 2 回)
- 60分 e-Learning + 確認テスト
- 内容:
  - シャドー AI の定義と例
  - リスク事例 (実際の業界事故)
  - 承認 AI ツールの使い方
  - 違反時の影響

### CS 要員追加 (年 1 回)
- 30 分研修 + ロールプレイ
- 通話中の AI 利用ルール

### 開発エンジニア追加 (入社時 + 年1回)
- コーディングAI使用ガイドライン
- Copilot Business のセキュリティ設定

## 違反検知

### 技術的検知
- **DLP (Data Loss Prevention)**: 個人情報を含むデータの外部AIサイト送信検知
  - Microsoft Purview DLP
  - 対象URL: chat.openai.com, claude.ai, gemini.google.com 等
- **CASB (Cloud Access Security Broker)**: Microsoft Defender for Cloud Apps
  - 未承認 SaaS への接続ブロック
  - "Shadow IT Discovery"
- **エンドポイント監視**: Microsoft Defender for Endpoint
  - クリップボード監視 (機密データを未承認サイトへペースト時アラート)

### 自己申告
- 社員が誤って利用した場合の報告窓口
- 自己申告は懲戒軽減

### 上司レビュー
- 1on1 で AI 利用状況確認
- 不審な業務効率改善があれば確認

## 4,500 名規模での展開課題

- **教育配信**: SAP SuccessFactors LMS (Layer 5 矢羽⑤) で全員配信
- **同意取得**: 入社時/契約更新時に必須
- **多言語**: 外国籍社員 (ベトナム/中国/ブラジル等) 向け翻訳版
- **協力会社**: 業務委託先にも適用 (契約書添付)
