# PERF-MEMORY-001: 対象システムにおいてメモリリーク（特に長寿命プロセス）が抑止され、計測値が閾値内に収まっているか

## 観点・確認内容

対象システムにおいてメモリリーク（特に長寿命プロセス）が抑止され、計測値が閾値内に収まっているか

## 適用条件

サーバ系

## OK基準

耐久試験24時間でメモリ増加率が閾値以下、リーク検出ツールで0件

## NG基準

メモリ増加率閾値超過、またはリーク1件以上

## 必要証跡

耐久試験レポート＋ヒーププロファイル
